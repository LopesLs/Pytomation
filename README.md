# Pytomation
Pytomation is a automation made with python for solve a particular problem of my job

## Setup Project

> On Windows:

```bash
py -m venv .venv
.venv/Scripts/activate
pip install -r requirements.txt
```

> On Linux:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
